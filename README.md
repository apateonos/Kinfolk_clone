## 사용한 기술

1. Front-End

- HTML / CSS
- JavaScript
- React
  - create-react-app settings
  - 공통 Component 재활용
  - React-slick
  - React-Router-DOM
  - fetch를 이용한 API 호출
  - map을 이용한 jsx 리턴
  - Routes.js 의 path 를 이용한 동적 라우팅
  - 함수형, 클래스형 컴포넌트 활용
  - Redux 적용
- SASS

2. Back-End

- Python
- Django
- MySQL

3. 공통

- JSON
- Git
- AWS
- Linux
- HTTP
- Postman
  - API 호출
  - 응답 확인
- Scrum 진행 방식을 통한 협업
